    在 Vue 单文件组件中，您可以使用以下配置属性来定义组件的行为和特性：

    1. data：

      • data 是一个函数，用于返回组件的初始数据对象。每个组件实例都会拥有自己的数据副本，确保数据的独立性。
      export default {
  data() {
    return {
      message: 'Hello, Vue!'
    };
  }
};

    2. props：

      • props 是一个数组或对象，用于接收父组件传递的数据。通过 props，父组件可以向子组件传递数据并进行通信。
      export default {
  props: ['title', 'description']
};

    3. computed：

      • computed 是一个包含计算属性的对象。计算属性是基于组件的状态或其他计算属性派生出来的属性，可以进行缓存，只在相关依赖发生变化时重新计算。
      export default {
  data() {
    return {
      firstName: 'John',
      lastName: 'Doe'
    };
  },
  computed: {
    fullName() {
      return this.firstName + ' ' + this.lastName;
    }
  }
};

    4. methods：

      • methods 是一个包含方法的对象。您可以在 methods 中定义组件的方法，用于响应事件或执行其他自定义逻辑。
      export default {
  methods: {
    greet() {
      console.log('Hello, Vue!');
    }
  }
};

    5. watch：

      • watch 是一个包含观察者的对象。通过 watch，您可以监听指定的数据变化并执行相应的操作。
      export default {
  data() {
    return {
      count: 0
    };
  },
  watch: {
    count(newValue, oldValue) {
      console.log('Count changed from ' + oldValue + ' to ' + newValue);
    }
  }
};

    这些是常见的配置属性，但还有其他更多的配置属性可用于定义组件的生命周期钩子、样式、组件间通信等。通过使用这些配置属性，您可以灵活地定义组件的行为和特性，并根据需要自定义和组合这些属性来满足您的需求。
