    Linux常用命令

    # Linux常用命令

    在线linux命令手册 (opens new window)

    # whoami

    **sudo xxx **以管理员用户执行xxx命令

    # 路径

    cd ~ 返回到根目录

    cd .. 返回到上一个目录

    pwd 显示当前路径

    ll 列表显示当前文件 和 权限

    ls

    ls -a 列出所有文件，包括隐藏文件

    # 网络

    ifconfig

    # 文件命令

    mkdir 文件夹名字 #创建文件夹

    rmdir 文件夹名字 #删除文件夹

    mv sourceFile /home/targetFile # 把当前目录下的sourceFile剪切到/home目录下并命名为targetFile

    rm #删除文件

    rm -r #删除文件夹

    # 权限

    chown [-r 递归] #更改所有者

    chown user -r myDir/ # 把文件夹myDir的所有者改成user

    chmod #更改文件权限

    文件权限有三类： 拥有者user，组group，其他用户other

    chmod 754 myDir/ # 当前用户（所有者）可读+写+执行，组group可读+执行，其他用户可读

    chmod g+w myDir/ # 为组group添加写write权限

    chmod -R 777 myDir/ # 给myDir/及其所有子目录添加所有权限

    ll # 可以查看当前目录下各子目录的文件权限

    # tar命令

    tar

    如果后缀是tar 就不用加

    z:后缀是tar.gz

    x : 解压

    c：压缩

    v：显示操作过程

    f: 指定文件名，f后面必须直接跟上要处理的文件名

    正确 ： tar -zxvf filename

    错误： tar -zxfv filename

    # zip文件

    zip

    unzip

    # vim编辑文件命令

    vi 文件名 默认进入到一般模式

    一般模式：

    上下左右移动光标

    yy 复制当前行

    5yy 复制5行

    p 粘贴

    dd 删除一行

    5dd 删除5行

    u 撤销

    i 或者 a 进入编辑模式

    Shift + : 进入命令行模式

    编辑模式：

    ESC进入一般模式

    命令行模式：

    q 退出

    q! 强制退出

    wq 保存并退出

    /关键字 搜索关键字 n下一个关键字

    set nu 显示行号

    set nonu 不显示行号

    # 管道： |

    将前一个命令的输出结果像管道一样传递给后一个命令作为输入

    ## 管道|用法

    ls | find -name myFile # 列出当前路径的文件（夹）并查找名字为“myFile”的，打印出来

    # grep： 按行查找并匹配

    ## grep用法

    grep -i mystring file.txt # 忽略大小写，在file.txt中查找并打印出有“mystring”的行

    ls -l | grep -i mystring # 在ls -l的输出中把有“mystring”的行打印出来（忽略大小写）

    ps：查看进程process status

    -e：等价于 ‘-A’ ，表示列出全部的进程

    -f：显示全部的列（显示全字段、端口、路径、等等）

    ## ps用法

    ps Aux # 查看系统所有的进程数据

    ps ax

    ps -ef | grep redis #查看redis

    # 输出重定向 >和>>

    >会覆盖目标的原有内容，当文件存在时，会先删除原文件，再重新创建文件，然后把内容写入该文件，否则直接创建文件。

    >>会在目标原有内容后追加内容，当文件存在时直接在文件末尾进行内容追加，不会删除原文件，否则直接创建文件。
