    Mybatis-XML写法

    # Mybatis-XML写法

    遇到复杂的sql语句 需要自己在dao层添加方法，在对应的xml文件手写sql语句。

    # #和$

    ​ MyBatis获取参数值的两种方式：${}和#{}

    ​ **#{}**的本质就是占位符赋值，将参数加上双引号，不会出现sql注入的问题

    ​ **${}**的本质就是字符串拼接(引号的方式)，直接把传进去的参数，原封不动加上去，如果有or 1=1这种语句，会导致有sql注入安全的问题

    ​ 尽量使用#{}

    List<User> getByNameLike(@Param("name") String name);

    <!--resultType为 返回类型，  全限定名字-->
<select id="getByNameLike" resultType="com.copa.entity.User">
    select * from t_user where name like "%"#{name}"%"
</select>

    # foreach循环

    • collection：设置要循环的数组或集合
    • item：表示集合或数组中的每一个数据
    • separator：设置循环体之间的分隔符，常用: , or
    • open：设置foreach标签中的内容的开始符 常用: (
    • close：设置foreach标签中的内容的结束符 常用: )

    # 批量添加

    mapper

    int insertMoreUser(@Param("userList") List<User> userList);

    xml

    <insert id="insertMoreUser">
		insert into t_user values
		<foreach collection="userList" item="user" separator=",">
			(null,#{user.name},#{user.remark},#{user.age},#{user.sex})
		</foreach>
</insert>

    # if

    mapper

    List<User> getUserByUserCondition1(@Param("user") User user);

    xml

    <select id="getUserByUserCondition1" resultType="com.copa.entity.User">
    select * from t_user where 1=1
    <if test="user.name != '' and user.name != null">
        and name = #{user.name}
    </if>
    <if test="user.age != '' and user.age != null">
        and age = #{user.age}
    </if>
    <if test="user.sex != '' and user.sex != null">
        and sex = #{user.sex}
    </if>
</select>

    # where

    where和if一般结合使用：

    a>若where标签中的if条件都不满足，则where标签没有任何功能，即不会添加where关键字

    b>若where标签中的if条件满足，则where标签会自动添加where关键字，并将条件最前方多余的and去掉

    **注意：**where标签不能去掉条件最后多余的and

    mapper

    List<User> getUserByUserCondition2(@Param("user") User user);

    xml

    <select id="getUserByUserCondition2" resultType="com.copa.entity.User">
    select * from t_user
    <where>
        <if test="user.name != '' and user.name != null">
            name = #{user.name}
        </if>
        <if test="user.age != '' and user.age != null">
            and age = #{user.age}
        </if>
        <if test="user.sex != '' and user.sex != null">
            and sex = #{user.sex}
        </if>
    </where>
</select>

    # trim

    添加or去除 前缀or后缀的某些内容

    **添加 ：**前后 (前prefix)(后suffix)

    **去除 ：**前后 (前prefixOverrides)(suffixOverrides)

    的某些内容

    List<User> getUserByUserCondition3(@Param("user") User user);

    <select id="getUserByUserCondition3" resultType="com.copa.entity.User">
    select * from t_user
    <trim prefix="where" suffixOverrides="and">
        <if test="user.name != '' and user.name != null">
            name = #{user.name} and
        </if>
        <if test="user.age != '' and user.age != null">
            age = #{user.age} and
        </if>
        <if test="user.sex != '' and user.sex != null">
            sex = #{user.sex}
        </if>
    </trim>
</select>
