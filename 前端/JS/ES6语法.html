    # ES6语法

    ES6（ECMAScript 2015）引入了许多令人兴奋的新语法和功能，提供了更强大和优雅的JavaScript编程体验。

    # 块级作用域和let/const关键字：

    ES6引入了块级作用域的概念，通过使用let和const关键字声明变量和常量。块级作用域允许我们更好地控制变量的作用范围，避免了变量声明提升的问题。下面是一个示例：

    {
  let x = 1;
  const y = 2;

  console.log(x); // 输出：1
  console.log(y); // 输出：2
}

console.log(x); // 报错：x未定义
console.log(y); // 报错：y未定义

    # 箭头函数：

    箭头函数是一种更简洁和直观的函数声明方式，它使用箭头（=>）来定义函数，并具有更方便的上下文绑定。下面是一个简单的箭头函数示例：

    // 传统函数声明
function add(a, b) {
  return a + b;
}

// 箭头函数
const add = (a, b) => a + b;

console.log(add(2, 3)); // 输出：5

    # 模板字符串：

    ES6的模板字符串允许我们在字符串中插入表达式和变量，使字符串拼接更加简洁和可读。下面是一个使用模板字符串的示例：

    const name = "Alice";
const age = 25;

const message = `My name is ${name} and I am ${age} years old.`;

console.log(message); // 输出：My name is Alice and I am 25 years old.

    # 解构赋值：

    解构赋值允许我们从数组或对象中提取值并赋给变量，提供了一种简洁的方式来操作和访问数据结构。下面是一个解构赋值的示例：

    // 数组解构赋值
const [a, b, c] = [1, 2, 

3];
console.log(a); // 输出：1
console.log(b); // 输出：2
console.log(c); // 输出：3

// 对象解构赋值
const person = { name: "Bob", age: 30 };
const { name, age } = person;
console.log(name); // 输出：Bob
console.log(age); // 输出：30

    # 默认参数：

    ES6允许我们在函数声明中指定参数的默认值，简化了函数的调用和定义。下面是一个默认参数的示例：

    function greet(name = "Guest") {
  console.log(`Hello, ${name}!`);
}

greet(); // 输出：Hello, Guest!
greet("Alice"); // 输出：Hello, Alice!

    演示Demo：计算阶乘 让我们通过一个简单的Demo来演示ES6的语法特性。我们将编写一个函数，使用箭头函数、默认参数和解构赋值来计算一个数字的阶乘。

    const factorial = (n, result = 1) => {
  if (n === 0) {
    return result;
  } else {
    return factorial(n - 1, result * n);
  }
};

const number = 5;
const result = factorial(number);

console.log(`The factorial of ${number} is ${result}.`);

    这个Demo使用了箭头函数来定义阶乘函数，使用默认参数来设置递归的初始值，通过解构赋值来简化函数的参数操作。运行上述代码，将会输出：The factorial of 5 is 120.

    # 模块化

    ​ ES6引入了官方的模块化系统，通过import和export关键字实现模块的导入和导出。本文将详细介绍ES6模块化的语法和使用方法，以及它与旧有模块方案（如CommonJS和AMD）的比较，帮助您充分利用模块化在JavaScript项目中实现可维护、可扩展的代码。

    # 导出和导入：

    ES6模块使用export关键字将模块中的变量、函数或类导出为公共接口，供其他模块使用。我们可以导出单个实体、默认导出或命名导出。下面是一些导出的示例：

    // 单个实体导出
export const PI = 3.1415;

// 默认导出
export default function add(a, b) {
  return a + b;
}

// 命名导出
export function multiply(a, b) {
  return a * b;
}

    其他模块可以使用import关键字来导入所需的模块。下面是一些导入的示例：

    // 导入单个实体
import { PI } from './math';

// 导入默认导出
import add from './math';

// 导入命名导出
import { multiply } from './math';

    # 默认导出和命名导出：

    ES6模块允许模块使用默认导出和命名导出。默认导出可以为模块指定一个默认的导出实体，而命名导出则允许模块导出多个实体。下面是默认导出和命名导出的示例：

    // math.js
const PI = 3.1415;

export default function add(a, b) {
  return a + b;
}

export { PI };

    // main.js
import sum, { PI } from './math';

console.log(PI); // 输出：3.1415
console.log(sum(2, 3)); // 输出：5
